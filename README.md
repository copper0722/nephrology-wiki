## 考試題型

- **Part A**：傳統考題（基礎）100 題
- **Part B**：臨床考題 50 題（部分含子題）
- **Part 2**：口試

## 使用方式

1. 直接閱讀 `.md` 檔案（GitHub 可預覽 Markdown）
2. **Clone 到本地，搭配你的 LLM 使用**（Claude/GPT/Gemini）— 推薦方式
3. 搜尋關鍵字：每個 claim 有 PMID/DOI，可直接查 PubMed
4. 需要 paywall 文章 → DM 我
