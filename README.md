## 使用方式

1. 直接閱讀 `.md` 檔案（GitHub 可預覽）
2. Clone 到本地，搭配你的 LLM 使用（Claude/GPT/Gemini）
3. 搜尋關鍵字：每個 claim 有 PMID，可直接查 PubMed
