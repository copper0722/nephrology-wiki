# Lupus Nephritis

## Exam Logic Summary

Lupus nephritis (LN) is among the highest-yield glomerular topics. The exam tests: (1) ISN/RPS classification and matching class to treatment intensity, (2) "full house" IF pattern recognition, (3) activity vs chronicity index interpretation, (4) serologic monitoring (anti-dsDNA, C3/C4), (5) treatment algorithm by class (KDIGO vs EULAR vs ACR). Common distractor: treating class V like class III/IV, or confusing activity (treatable) with chronicity (irreversible).

**Textbook ref:** Brenner Ch. 31 (Lupus Nephritis), Harrison Ch. 313 (SLE), KDIGO 2024 Glomerular Diseases

---

## Epidemiology

- 20%-60% of SLE patients develop LN during lifetime
- 25%-50% present with kidney disease at SLE diagnosis
- More common in: males with SLE, juvenile-onset SLE, Black (34%-51%), Asian (33%-82%), Hispanic (31%-49%) vs White (14%-23%)
- Biopsy-proven LN prevalence: 29% overall (systematic review, 19 studies)
- Class IV most common (28%-48%), followed by class III (11%-24%), class V (12%-20%)

## ISN/RPS Classification (2003 + 2018 Working Group Revisions)

| Class | Name | Histology | Key features |
|---|---|---|---|
| I | Minimal mesangial | Normal LM; mesangial immune deposits on IF only | Incidental finding |
| II | Mesangial proliferative | Mesangial hypercellularity (>=4 nuclei) + mesangial deposits | Mild; isolated subepithelial/subendothelial deposits possible |
| III | Focal | Endo/extracapillary GN involving < 50% glomeruli | Subdivided: A (active), A/C (active+chronic), C (chronic scars) |
| IV | Diffuse | Endo/extracapillary GN involving >= 50% glomeruli | Most severe proliferative; wire loops, crescents; 2018: removed IV-S/IV-G subdivisions |
| V | Membranous | Subepithelial deposits (global/segmental) | May coexist with III or IV (both diagnosed); resembles primary MN |
| VI | Advanced sclerosis | >= 90% globally sclerosed, no residual activity | End-stage; no immunosuppressive benefit |

**Exam pearl:** Class III and IV = PROLIFERATIVE = aggressive immunosuppression required. Class V pure = treat like nephrotic syndrome unless nephritic features present. Class I/II = treat extrarenal SLE only.

### 2018 Working Group Key Changes

- "Endocapillary hypercellularity" replaces "endocapillary proliferation"
- Crescent defined as extracapillary hypercellularity involving >= 10% of Bowman capsule
- Cellular crescent: > 75% cells/fibrin, < 25% fibrous matrix
- Fibrous crescent: > 75% fibrous matrix, < 25% cells/fibrin
- Fibrocellular: 25%-75% cells and fibrin
- Modified NIH activity and chronicity scoring replaces active/chronic/active-chronic subdivisions
- Class IV subdivisions (IV-S, IV-G) removed

## Activity and Chronicity Indices (NIH Modified)

### Activity Index (reversible, treatable)

| Feature | Score |
|---|---|
| Cellular proliferation | 0-3 |
| Fibrinoid necrosis/karyorrhexis | 0-3 (x2) |
| Cellular crescents | 0-3 (x2) |
| Hyaline thrombi/wire loops | 0-3 |
| Leukocyte infiltration | 0-3 |
| Interstitial inflammation | 0-3 |
| **Maximum** | **24** |

### Chronicity Index (irreversible, poor prognosis)

| Feature | Score |
|---|---|
| Glomerular sclerosis | 0-3 |
| Fibrous crescents | 0-3 |
| Tubular atrophy | 0-3 |
| Interstitial fibrosis | 0-3 |
| **Maximum** | **12** |

**Exam pearl:** High activity index = indication for aggressive immunosuppression. High chronicity index = poor response to treatment, irreversible damage. A patient with high chronicity + low activity = no benefit from intensifying immunosuppression.

## Immunofluorescence — "Full House" Pattern

Positive staining for IgG, IgA, IgM, C1q, C3, and C4 = "full house" = virtually pathognomonic for lupus nephritis.

**Exam pearl:** "Full house" IF is the single most distinctive biopsy finding for LN. No other GN shows simultaneous staining for all 3 Ig classes + complement. If an exam question describes full house IF, the answer is lupus nephritis regardless of other information.

## Diagnostic Workup

### When to Biopsy (Strong Recommendation)

- Proteinuria >= 0.5 g/24h OR spot UPCR > 0.5 g/g
- Active urine sediment (RBC, WBC, cellular casts)
- Unexplained decrease in GFR
- Biopsy reveals pathology, guides treatment, identifies coexisting conditions (TMA in up to 25%)

### Key Serologies

| Test | Clinical significance |
|---|---|
| Anti-dsDNA | Rises before flare; 100% of renal exacerbations have positive anti-dsDNA |
| C3/C4 | Low = active disease; fluctuating complement predicts renal involvement |
| Anti-Sm | Highly specific for SLE (not for nephritis severity) |
| Anti-C1q | Sensitivity 58%, specificity 75% for LN (limited by heterogeneity) |
| ANA | Entry criterion; >= 1:80 on HEp-2 |
| Antiphospholipid antibodies | Assess thrombotic risk |

**Exam pearl:** Rising anti-dsDNA + falling C3 = impending renal flare. HR 3.4 for flare with anti-dsDNA >= 75 units/mL; HR 0.27 (protective) with C3 < 75 units/mL at baseline (i.e., already low = active disease present, not predicting NEW flare).

---

## Non-LN Kidney Disease in SLE (approximately 5%)

- Lupus podocytopathy (MCD or FSGS pattern, 1%-2%) - foot process effacement without subendothelial deposits
- Tubulointerstitial nephritis
- Thrombotic microangiopathy (up to 25%)
- Antiphospholipid-associated nephropathy

**Exam pearl:** SLE patient with nephrotic syndrome + normal LM + foot process effacement on EM = lupus podocytopathy (NOT class V). Treat as MCD with steroids.

## Treatment by ISN/RPS Class

### Class I/II

- Treat extrarenal SLE manifestations
- Low-dose corticosteroid + another immunosuppressive agent for maintenance
- If nephrotic syndrome present: evaluate for lupus podocytopathy, treat as MCD

### Class III/IV (+/- V) — Proliferative LN (KDIGO 2024)

**Induction (total initial + maintenance >= 36 months):**

Corticosteroids: methylprednisolone 0.25-0.5 g/day IV pulse x 3 days, then prednisone 0.3-1 mg/kg/day orally tapered to <= 5 mg/day by 3-6 months

PLUS 1 of the following (KDIGO Grade 1, Level B):

| Option | Regimen |
|---|---|
| MPAA (preferred) | MMF 500 mg BID, titrate to 1.5 g BID |
| Low-dose CYC (Euro-Lupus) | 500 mg IV q2w x 3 months OR 0.5-1 g/m2 IV monthly x 6 months |
| MPAA + CNI | For GFR > 45 mL/min/1.73 m2 |
| Belimumab + MPAA or low-dose CYC | 10 mg/kg IV q2w x 3, then q4w (or SC 400 mg weekly x 4, then 200 mg weekly) |

**Alternative (3-drug regimen for select patients):**

CNI (tacrolimus 0.05 mg/kg/day, trough 4-6 ng/mL; OR voclosporin 23.7 mg BID; OR cyclosporine 3-5 mg/kg/day, trough 60-100 ng/mL) + reduced-dose MPAA (MMF 1 g/day) + corticosteroids

Consider 3-drug regimen for patients who:
- Cannot tolerate standard-dose MPAA
- Refuse cyclophosphamide
- Need rapid proteinuria reduction
- Are of Chinese descent (reportedly higher efficacy)

**Exam pearl:** Voclosporin does NOT require therapeutic drug monitoring (unlike tacrolimus/cyclosporine). Requires GFR >= 45 mL/min/1.73 m2 and concurrent MPAA.

**Maintenance:**

- MPAA first choice (KDIGO Grade 1, Level B): MMF 750-1000 mg BID early, goal 1-2 g/day
- Azathioprine 1.5-2 mg/kg/day: alternative; PREFERRED for patients planning pregnancy
- If triple therapy used for induction with belimumab/CNI, may continue same regimen
- Taper corticosteroids to lowest possible dose (prednisone < 5-7.5 mg/day); consider discontinuation after >= 12 months complete response

### Pure Class V

- Low-level proteinuria: HCQ + RAS blockade + BP control; immunosuppression based on extrarenal SLE
- Nephrotic syndrome: corticosteroid + 1 other drug (MPAA, CYC, CNI, rituximab, azathioprine)
- Initial immunosuppression: MPAA (2-3 g/day) + CNI + corticosteroids (Conditional Recommendation)

### Refractory/Relapse Management

- Confirm adherence + check drug levels
- Repeat biopsy: assess class change, chronicity, TMA
- Switch to alternative first-line regimen
- Add rituximab (1 g IV days 1 and 15) for inadequate response
- Escalate to triple therapy or extended CYC IV
- ACR 2025: for refractory disease after 2 standard courses, consider anti-CD20, combination of 3 non-corticosteroid immunosuppressives, or clinical trial

## Key Trials

| Trial | Year | N | Bottom line |
|---|---|---|---|
| Euro-Lupus Nephritis Trial | 2004 | 90 | Low-dose IV CYC (500 mg q2w x 6) equivalent to high-dose (monthly x 6) at 73 months; permanent kidney impairment 20% vs 23% (NS) |
| Euro-Lupus 10-year follow-up | 2010 | 84 | No difference in death, doubling Cr, or ESKD at 10 years |
| Cochrane 2018 (CYC vs steroids) | 2018 | 5,175 | IV CYC + steroids: RR 0.59 for doubling Cr (NNT 5-21); RR 0.23 for renal relapse (NNT 3-6); increased ovarian failure RR 2.18 |
| Cochrane 2018 (MMF vs IV CYC) | 2018 | 868 | MMF: similar complete remission (RR 1.17 NS) but less alopecia (RR 0.29), vomiting (RR 0.48), nausea (RR 0.48); more diarrhea (RR 2.42) |
| AURORA (voclosporin) | 2021 | 357 | Voclosporin + MMF: higher complete renal response vs placebo + MMF at 52 weeks |

**Exam pearl:** MMF vs IV CYC for induction = similar efficacy but MMF has fewer adverse effects (less alopecia, less nausea, less ovarian failure). This is a commonly tested comparison. CYC preferred only when fertility is not a concern or adherence to oral regimen is questionable.

## Adjunctive Therapy (All LN Patients)

| Intervention | Strength |
|---|---|
| Hydroxychloroquine | Strong Recommendation (all LN patients) |
| ACEi/ARB for proteinuria/HTN | Strong Recommendation |
| Statin for dyslipidemia/CVD risk | Conditional Recommendation |
| Aspirin for antiphospholipid antibodies | Conditional Recommendation (weigh bleeding risk) |
| Vitamin D + calcium + antiresorptive | Conditional Recommendation |
| Nonlive vaccines | Conditional Recommendation |
| Anticoagulation if albumin < 20 g/L + nephrotic | Conditional Recommendation |

**Exam pearl:** HCQ is the ONLY drug recommended for ALL LN patients regardless of class. It reduces flares, improves survival, has antithrombotic properties. Stopping HCQ increases flare risk.

## Pregnancy in Lupus Nephritis

- Azathioprine (2 mg/kg/day) = preferred maintenance for patients planning pregnancy (Strong Recommendation)
- CNI (tacrolimus) acceptable during pregnancy
- CONTRAINDICATED in pregnancy: MMF/MPA (teratogenic), cyclophosphamide, methotrexate
- Continue HCQ throughout pregnancy
- Stable remission >= 6 months before conception is ideal
- Monitor: anti-dsDNA, complement, proteinuria, BP monthly
- Risk: preeclampsia, flare, fetal loss (especially with antiphospholipid antibodies)

- SLEDAI-2K: 24 weighted clinical/lab features over 9 organ systems; score >= 6 clinically important
- Renal domain in EULAR/ACR 2019: proteinuria > 0.5 g/24h = 4 points; class II/V biopsy = 8 points; class III/IV biopsy = 10 points
- Anti-dsDNA = 6 points (SLE-specific antibody domain)

## High-Yield Exam Scenarios

1. **Young woman with nephrotic syndrome, low C3/C4, positive anti-dsDNA, full house IF on biopsy:** Class IV or V lupus nephritis. If proliferative features present = Class IV. Treatment: steroids + MPAA or CYC + HCQ.

2. **SLE patient on MMF develops rising Cr + active sediment:** Renal flare. Check anti-dsDNA (likely rising), C3/C4 (likely falling). Consider repeat biopsy to assess class change. Options: same therapy at higher dose, switch to alternative first-line, add rituximab.

3. **SLE patient with nephrotic proteinuria but NORMAL biopsy on LM:** Lupus podocytopathy (1%-2% of SLE kidney disease). EM shows foot process effacement without subendothelial deposits. Treat as MCD.

4. **SLE patient planning pregnancy on MMF:** Switch to azathioprine + continue HCQ. MMF is teratogenic. Wait >= 6 weeks after MMF discontinuation before conception.

5. **Biopsy shows >= 90% global sclerosis with no active lesions:** Class VI. No benefit from immunosuppression. Plan for RRT.

6. **SLE patient with TMA on biopsy (25% of LN):** May occur with or independently of LN. Check ADAMTS13 (normal in SLE-TMA vs < 10% in TTP). Manage coexisting TMA per guidelines.

7. **Exam asks about "wire loops":** Subendothelial immune deposits so massive they thicken the capillary wall = wire loop lesion. Pathognomonic for class IV LN with high activity.
